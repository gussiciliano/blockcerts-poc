# Blockcerts PoC

PoC Emisor + Verificador (simulado anchoring) en Node.js/TypeScript.

Quick start

1. Install deps

```bash
cd /Users/gustavosiciliano/unq
yarn
```

2. Generate keys

```bash
yarn gen-keys
```

3. Seed example credential

```bash
yarn seed
```

4. Start server

```bash
yarn start
```

Open `http://localhost:3000/public` to use the verification UI.
